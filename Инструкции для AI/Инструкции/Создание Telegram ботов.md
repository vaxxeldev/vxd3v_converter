ДОПОЛНИТЕЛЬНО:

1. ЦВЕТНЫЕ КНОПКИ
   Reply и Inline-кнопки поддерживают свойство "style", которое может принимать одно из трёх значений (не считая дефолтного):
   🔴 danger — красная кнопка (идеально для «Удалить», «Отмена», «Бан»).
   🟢 success — зелёная кнопка («Подтвердить», «Оплатить», «Принять»).
   🔵 primary — синяя кнопка (для основных целевых действий).
   ⚪️ Стандартный — привычный прозрачно-серый цвет.

2. ПЛАВНЫЙ ВЫВОД СООБЩЕНИЙ
   Метод "sendMessageDraft" - Это эффект плавной генерации текста по буквам, как у ChatGPT и других нейросетей.
   Раньше это работало только для ботов в чатах с включёнными темами, но теперь настоящий 
   релиз снимает все ограничения! Метод "sendMessageDraft" разрешили использовать абсолютно 
   всем ботам и в любых чатах (Telegram Bot API 9.5, подробнее: https://core.telegram.org/bots/api#sendmessagedraft). Если мы пишем ИИ-бота или
   просто хотим сделать вывод больших текстов более интерактивным - можем использовать этот метод, по необходимости.

3. КАСТОМНЫЕ ЭМОДЗИ
   Метод "icon_custom_emoji_id" - теперь на кнопки/сообщения бота можно
   добавить кастомные эмодзи (внимание на файл premium_emoji.json).
